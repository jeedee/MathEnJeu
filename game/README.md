# MathEnJeu

Jeu multijoueur où on répond à des questions mathématiques pour progresser.

## Project setup

```
npm install
```

### Compiles and hot-reloads for development (works on port 8080)

```
npm run dev
```

### Compiles and run for docker (works on port 8080)

```
- docker-compose build

- docker-compose up
```
